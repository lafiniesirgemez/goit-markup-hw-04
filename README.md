## ✅ Requirements Checklist

### Project

- [x] All images stored in `images/` folder
- [x] CSS styles placed in `css/styles.css`
- [x] File names lowercase, no spaces or special characters
- [x] Code formatted with **Prettier**
- [x] `modern-normalize` included
- [x] Project follows semantic and accessible HTML guidelines

### Markup

- [x] All icons combined into `icons.svg` sprite
- [x] Icons embedded via `<svg>` + `<use>`
- [x] Semantic HTML5 (`header`, `main`, `section`, `footer`, `nav`, `ul`, `li`)
- [x] Accessible attributes (`aria-label`, `aria-hidden`)
- [x] Google Fonts included (`Raleway`, `Roboto`)

### Design

- [x] Hero section with gradient + image background
- [x] Features section with styled icon cards
- [x] Team section with member cards + permanent shadows
- [x] Portfolio cards with hover overlay + shadow
- [x] Footer with logo and social icons
- [x] Navigation active state underlined with `::after`
- [x] Smooth hover/focus transitions (`250ms cubic-bezier(0.4, 0, 0.2, 1)`)
